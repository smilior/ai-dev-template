HOW TO USE

使い方

開発はこの3手で回る。あなたが書くのはチャットの指示だけ。

  1. 題材を伝える。「◯◯を管理するシステムを作りたい」で企画が動き出す
  2. 確認事項に答える。エージェントが業務ルールを勝手に決めず、質問リストにして返してくる
  3. PR をマージする。マージ = 承認記録。docs/ とコードが1段ずつ育つ
  • チーム名鑑 → 指示者と10体の専門エージェント。誰に何を頼めるか、起動フレーズ付き。 STD-AIDEV-002 ・ agents-roster.html
  • 案件書類フォルダ → 雛形を埋めながら進める作業場。全書類に記入ガイドと記入例入り。 docs/project/
